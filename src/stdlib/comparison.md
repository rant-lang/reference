# Comparison

## eq

```rant

[%eq: lhs; rhs]

```
&rarr; `bool`

Returns `@true` if `lhs` and `rhs` are equal.

### Parameters

**`lhs`** &larr; `any` <br/>
The left-hand operand of the comparison.

**`rhs`** &larr; `any` <br/>
The right-hand operand of the comparison.
